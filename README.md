# python-flask-hyakunitigonoboku

100日後の設定でしたが、やっぱ10年後にしました。

## 使い方

いくつかの項目を選択することであなたの10年後を占います。

1年ごとの物語形式となっています。

*なお、本アプリはGPT APIを使用するため、利用の場合はAPIキーの取得が必要です*
